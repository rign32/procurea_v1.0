## Copy angle (1 sentence)
Not a thought-leadership essay — an operator's playbook with a 90-day plan, country-category fit matrix, and a cost model that survives CFO review.

## Must-include elements (in the asset itself)
- **Page 1**: executive summary (who this is for, 3 things you'll get out of it).
- **Page 2**: the 5-criteria category prioritization (which categories to move first, which never).
- **Page 3**: country-category fit matrix (EU-CEE / Portugal / Turkey / Mexico × 7 categories: plastics, metals, electronics, textiles, packaging, chemicals, machining).
- **Page 4-5**: the 90-day pilot plan, week-by-week, with named outputs per week.
- **Page 6**: financial model template — total cost of diversification (year 1 tooling + audit + higher unit cost) vs. risk-adjusted single-country cost. Worked example.
- **Page 7**: risk register template — labor cost inflation, currency, CBAM, capacity ramp.
- **Page 8**: anonymized pilot-cohort snapshot (composite: "an automotive tier 2 client moved 2 categories from China to Poland + Turkey, 90 days, X% landed-cost delta").
- Clean design: 2-column layout where useful, data tables, no stock photos.
