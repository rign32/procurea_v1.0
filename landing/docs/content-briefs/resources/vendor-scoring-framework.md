## Copy angle (1 sentence)
A 10-criteria scorecard structured in 3 tiers (universal / compliance / strategic), with anchored 1-5 descriptions that reduce scoring subjectivity between analysts.

## Must-include elements (in the asset itself)
- **PDF page 1**: the 10 criteria overview + the 3-tier structure (Universal 5 / Compliance 3 / Strategic 2).
- **PDF page 2**: anchored 1-5 rubric for each criterion (what a "3" looks like vs. a "5"). This is the differentiator.
- **PDF page 3**: weighting model (defaults per category archetype — commodity, strategic, service, regulated), calibration method (2 analysts, same suppliers, gap < 1.5).
- **PDF page 4**: audit-trail template — how to document the "why" behind each score so Internal Audit can review.
- **XLSX tab 1 — Scorecard**: 10 criteria rows × N suppliers columns, weighted sum formula.
- **XLSX tab 2 — Calibration**: two-analyst gap check to flag criteria needing more rubric work.
