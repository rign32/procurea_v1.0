## Copy angle (1 sentence)
Not another ISO 31000 diagram — a practitioner's 12 checks across 5 risk categories, with the registry URL, the data source, and the re-check cadence for each.

## Must-include elements (in the asset itself)
- Page 1: executive overview + the 5 risk categories (Financial, Operational, Geopolitical, Regulatory, ESG).
- Page 2: the 12 checks — each with a 1-line description, the data source, the verification URL, and "what a red flag looks like."
- Page 3: risk scoring matrix (how to convert 12 checks into a single 0-100 score, with worked example).
- Page 4: re-check cadence template (monthly / quarterly / annual tasks) + a 1-page printable version of the checklist for per-supplier use.
- Branded but NOT marketing-heavy; this is a reference document. CTA on last page only.

## Format
PDF, A4, 4 pages, editable print-friendly. Procurea branded footer with `procurea.io/resources/supplier-risk-checklist` and date-of-publication.

## Value props (5 bullets for landing page)
- 12 checks across Financial, Operational, Geopolitical, Regulatory, and ESG categories.
- Each check names the registry or data source to use (VIES, OFAC, D&B, ISO certification databases).
- Scoring rubric converts 12 findings into one risk score for board reporting.
- Includes a 2026-specific section: CBAM, CSRD, Red Sea shipping, US Section 232 exposure.
- 1-page printable version for use in supplier onboarding reviews.
