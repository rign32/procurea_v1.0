## Copy angle (1 sentence)

## Must-include elements (in the asset itself)
- 3 pre-filled examples: machined part (5 suppliers, EUR), event catering (4 suppliers, EUR), construction subcontractor (6 suppliers, PLN).
- A weighting tab: 10 criteria with adjustable weights (default 40/25/20/15 across price/lead-time/quality/terms).
- Formulas that auto-rank suppliers and flag incomplete rows.
- A "deviations" column with conditional formatting.
- A cover-letter DOCX that a buyer can use to send suppliers a structured RFQ (placeholders for product spec, MOQ, lead time, certifications required, deadline).
- Notion version: database with the same columns, formula property for weighted score, views for "received / awaiting / declined."
- A 1-page "how to use" PDF (front page of the zip) that takes 2 minutes to read.
